# Guessr Game List

A curated list of "guess the place / time" games and other geography puzzles.

## Guessr-style games
- GeoGuessr - https://www.geoguessr.com/
- TimeGuessr - https://timeguessr.com/
- WhenTaken - https://whentaken.com/
- WhereTaken - https://wheretaken.com/
- OpenGuessr - https://geoguessrfree.io/openguessr



## Other geography games
- Travle.Earth - https://travle.earth/
- Flagle - https://www.flagle.io/
- Globle - https://globle.org/?practice_mode=true
- Worldle - https://theworldle.com/
